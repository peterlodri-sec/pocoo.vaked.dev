What is GlyphCSS?

GlyphCSS renders 3D models as ASCII art in a single <pre> element. No WebGL. No canvas. Every character is inspectable, hoverable, clickable. It's the most beautiful abuse of the DOM we've seen since marquee tags.

🎮 OBJ, glTF, GLB, STL, VOX

Load any 3D format. GlyphCSS rasterizes polygons into a character grid. One string to textContent per render.

🖱 Interactive Hotspots

Place <glyph-hotspot> at any 3D coordinate. Real DOM events, :hover styles, role="button" accessibility.

⚛ React · Vue · Vanilla

Custom elements for vanilla JS. Wrappers for React and Vue. Even works in the terminal.

GlyphCSS Research Findings

🎯 Braille Palette

glyphPalette="braille" — 8 dots per cell, gap-free, highest resolution. Built-in, zero config. The fastest path to production-quality renders.

🔧 Block Gap Fix

text-shadow: 0.5px 0 0 currentColor on the <pre> eliminates gaps between block characters. 1-line CSS fix.

⚡ 40% Faster

Cross-frame shading cache (PR #10) caches shade values between frames. Significant perf boost for animations.

🔤 Font Extrusion

PR #13 adds Google Fonts → 3D ASCII extrusion. Any web font becomes a 3D glyph mesh.
